6. 理解话题

运行小乌龟模拟程序

  •   $ sudo apt-get install ros-<distro>-rqt 
  $ sudo apt-get install ros-<distro>-rqt-common-plugins

  • ros话题介绍

    • rostopic bw display bandwidth used by topic

    • rostopic echo print messages to screen 可以显示在某个话题上发布的数据。

    • rostopic pub publish data to topic 可以把数据发布到当前某个正在广播的话题上

      • 双破折号告诉命令选项解析器接下来的参数部分都不是命令选项

    • rostopic type print topic type 用来查看所发布话题的消息类型。

      • rosmsg命令来查看消息的详细情况

  • rosparam get 获取参数

  • rosparam delete 删除参数

  • rosparam list 列出参数名

8. 使用rqt_console与roslaunch

使用rqt_console和rqt_logger_level

  • rqt_console属于ROS日志框架(logging framework)的一部分，用来显示节点的输出信息。
  • rqt_logger_level允许我们修改节点运行时输出信息的日志等级（logger levels）
    • Fatal是最高优先级，Debug是最低优先级

使用roslaunch

  • roslaunch可以用来启动定义在launch文件中的多个节点。
